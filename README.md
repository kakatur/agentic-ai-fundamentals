# Agentic AI Fundamentals

A comprehensive, free video course teaching you to build production-ready AI agents from scratch.

## 🌐 Live Site

**Main Site:** [https://agenticaifundamentals.com](https://agenticaifundamentals.com)  
**GitHub Pages:** [https://kakatur.github.io/agentic-ai-fundamentals/](https://kakatur.github.io/agentic-ai-fundamentals/)  
**YouTube Channel:** [@AgenticAIFundamentals](https://youtube.com/@AgenticAIFundamentals)

## 📚 What You'll Learn

Master the complete journey from Python basics to production-ready multi-agent systems:

### 15 Comprehensive Modules

1. **Python Fundamentals for AI** - Variables, OOP, async programming, FastAPI
2. **AI & LLM Fundamentals** - Neural networks, tokenization, embeddings, model selection
3. **Prompt Engineering & LLM APIs** - Advanced prompting, structured outputs, cost optimization
4. **Vector Databases & Embeddings** - Semantic search, FAISS, Pinecone, Weaviate
5. **Building RAG Systems** - Document ingestion, chunking, hybrid retrieval, evaluation
6. **Function Calling & Tools** - Tool design, MCP, error handling, security
7. **Single Agent Systems** - ReAct pattern, LangChain, memory, human-in-the-loop
8. **Context Engineering & Agent Memory** - Context windows, prompt boundaries, memory layers, compression, caching, and privacy
9. **Multi-Agent Systems** - LangGraph, orchestration patterns, agent-as-tool
10. **Security & Guardrails** - Prompt injection defense, three-layer architecture, semantic routing
11. **Agent Testing & Evaluation** - Unit tests, simulation environments, LLM-as-judge
12. **Streaming & Real-Time Agents** - SSE, WebSockets, bidirectional communication
13. **LLMOps & Observability** - Tracing, cost tracking, A/B testing, feedback loops
14. **Cloud Deployment & Infrastructure** - AWS, Docker, ECS Fargate, Bedrock
15. **Hands-On Projects** - 8+ real-world projects at every skill level

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/kakatur/agentic-ai-fundamentals.git
cd agentic-ai-fundamentals

# Start a local server
python3 -m http.server 8000

# Open in browser
open http://localhost:8000
```

Or use VS Code Live Server extension.
